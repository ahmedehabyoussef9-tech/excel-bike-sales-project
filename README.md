# Bike Sales Analysis (Excel Dashboard)

## Overview

This project analyzes customer data to identify key factors influencing bike purchase behavior. The goal is to uncover patterns in demographics and lifestyle attributes that affect buying decisions.

---

## Tools Used

* Microsoft Excel
* Pivot Tables
* Slicers
* Charts and Dashboard Design
* Data Cleaning Techniques

---

## Data Preparation

The dataset was cleaned and prepared before analysis:

* Removed inconsistencies and unnecessary spaces
* Standardized categorical values
* Created calculated fields using nested IF statements
* Structured data for Pivot Table analysis

---

## Analysis Performed

The following factors were analyzed to understand their impact on bike purchases:

* Income level
* Age group
* Commute distance
* Car ownership
* Number of children
* Home ownership
* Demographics (marital status, region, education)

---

## Dashboard Features

* Interactive dashboard with slicers (Marital Status, Region, Education)
* Multiple visualizations for comparing buyer vs non-buyer behavior
* Dynamic filtering for deeper exploration

---

## Key Insights

* Customers with shorter commute distances show a higher likelihood of purchasing bikes, suggesting bikes are used for short-distance commuting.
* Home ownership does not show a strong relationship with bike purchases, indicating it is not a major influencing factor.
* Middle-aged customers represent the highest proportion of bike buyers.
* Higher income is associated with increased bike purchase likelihood.
* Customers with fewer cars are more likely to purchase bikes, suggesting bikes may substitute for car usage.
* Customers without children tend to purchase bikes more often than those with children.

---

## Project Structure

* `Raw_Data` → Original dataset
* `Working Sheet` → Data cleaning and transformations
* `Pivot Table` → Aggregated analysis
* `Dashboard` → Final interactive dashboard

---

## Conclusion

This analysis highlights how demographic and lifestyle factors influence bike purchasing behavior. The findings can be used to guide targeted marketing strategies and better understand customer segments.
